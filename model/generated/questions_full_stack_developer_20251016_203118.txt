Q2 [coding]: Write a JavaScript function to calculate the sum of all numbers in an array.
Starter code:
function sumArray(arr) {
Answer: return arr.reduce((acc, curr) => acc + curr, 0);

Q3 [debug]: The following Python code is intended to check if a number is even. Identify and fix the bug.
Buggy code:
def is_even(num):
  if num % 2 == 1:
    return False
  else:
    return True
Fix:
def is_even(num):
  return num % 2 == 0
